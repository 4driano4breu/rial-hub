Como criar e gerenciar usuários

Disponível apenas para ADMIN e SUPERADMIN. Acesse /admin/.

  1. No painel Admin, clique em Usuários
  2. Clique em + Novo Usuário
  3. Preencha nome, e-mail, senha e cargo
  4. Clique em Criar — o usuário já pode fazer login

Para desativar um usuário sem excluí-lo, edite o perfil e desmarque "Ativo".

Como alterar configurações da organização

Acesse /admin/ e clique em Configurações. Parâmetros disponíveis:

  • Alíquotas — percentuais de INSS, IR e ISS usados no cálculo de notas
  • CAP AEGEA / Guariroba — toneladas contratadas para cada contrato
  • Contratos ativos — lista de contratos disponíveis para seleção nas notas
  • Google Sheets ID — planilha de origem para sincronização automática da Usinagem

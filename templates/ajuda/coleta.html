Como criar um formulário

Acesse /formularios/ no menu "📋 Coleta de Campo" e clique em + Novo Formulário.

  1. Digite o nome do formulário (ex: "Abastecimento Diário")
  2. Adicione uma descrição opcional
  3. Marque "Aceita resposta sem login?" se operadores não tiverem conta no OBRIA
  4. Clique em Criar e adicionar campos

Como adicionar campos

Após criar, você é levado ao editor de campos. Tipos disponíveis:

  • Texto curto — campo livre de uma linha (ex: placa do veículo)
  • Texto longo — área de texto (ex: observações)
  • Número — aceita valores numéricos com unidade (ex: 45 litros)
  • Seleção — lista de opções predefinidas (ex: motorista)
  • Data / Hora — picker nativo do celular
  • Foto — abre a câmera do celular
  • Sim/Não — dois botões de escolha rápida
  • GPS — captura a localização automaticamente

Para cada campo: escolha o tipo, escreva o rótulo e marque "obrigatório" se necessário.

Como compartilhar o link

Na tela de edição do formulário, o link mobile aparece automaticamente. Copie e envie por WhatsApp, ou imprima um QR code apontando para esse link e cole na máquina ou local de coleta.

O operador abre o link no celular e preenche — sem instalar nada, sem fazer login (se configurado como anônimo).

Como ver e exportar respostas

  1. Acesse /formularios/ e clique no formulário desejado
  2. Clique em 📊 Respostas
  3. Veja a tabela com todas as respostas recebidas
  4. Clique em Exportar CSV para baixar os dados em planilha

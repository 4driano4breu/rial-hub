Como cadastrar uma máquina

Cada equipamento da frota precisa estar cadastrado para receber checklists.

  1. Acesse /equipamentos/ pela barra superior (🚜 Equipamentos).
  2. Clique em + Nova Máquina.
  3. Informe o nome, o tipo e o modelo.
  4. Salve para a máquina aparecer na frota.

Como criar um template de checklist

O template define quais itens o operador vai verificar todos os dias.

  1. Na página de equipamentos, abra Templates.
  2. Crie um novo template e dê um nome.
  3. Adicione os itens a verificar (fluidos, freios, pneus, etc.).
  4. Marque quais itens são obrigatórios.
  5. Salve o template e vincule-o à máquina.

Como o operador acessa pelo celular

O operador preenche o checklist direto no celular, sem precisar de senha.

  1. Cada máquina tem um link no formato /equipamentos/m/{id}.
  2. Compartilhe o link com o operador (ou um QR Code, se disponível).
  3. O operador abre o link, responde os itens e envia.
  4. O checklist enviado aparece no detalhe da máquina.

Como interpretar o streak e o heatmap

Esses indicadores mostram a regularidade dos checklists.

  1. O streak é o número de dias seguidos com checklist em dia.
  2. O heatmap é um calendário colorido: cada quadrado é um dia.
  3. Dias preenchidos ficam destacados; dias sem checklist ficam vazios.
  4. Use isso para ver máquinas que estão sendo esquecidas.
